Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates and names a new branch and switches to it
  • git branch -d branch-name: deletes a branch; will not delete if content is not added to main branch yet
  • git add -A: commits all files in the working branch and stages them to be moved
  • git commit -m: commits above mentioned files to the repository; the flag -m adds a message to the commit about changes implemented
  • git pull origin main (or other branch-name): pulls any updated code from the main branch (or other branch) into the feature branch; important to do before adding changes
  • git push origin branch-name: pushes changes to GitHub, making it possible to open a PR (pull request) so that, after peer reviews, a branch can be merged to the main branch
